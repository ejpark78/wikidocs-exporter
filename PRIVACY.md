## English

### Data Collection and Usage

WikiDocs Exporter is a Chrome extension that helps users export content from wikidocs.net to their local note-taking applications (Obsidian or Joplin).

### What Data We Collect

This extension processes the following data locally on your device:

- **WikiDocs Content**: Chapter titles, body text, and images from wikidocs.net pages
- **User Settings**: Export options (target app, image inclusion, frontmatter, etc.) stored in Chrome local storage
- **Joplin API Token**: Stored locally for Joplin integration (only when user explicitly saves it)

### How We Use Data

- **Local Processing Only**: All data processing happens locally on your device. No data is sent to external servers.
- **No Third-Party Services**: We do not use any third-party cloud services, analytics, or tracking.
- **Export Purpose Only**: Data is used solely for exporting content to your local note apps.
- **No Collection**: We do not collect, transmit, or store any personal information externally.

### Data Storage

- User settings are stored in Chrome's local storage (`chrome.storage.local`)
- Joplin API token is stored locally only with explicit user consent
- All exported data remains on your local device

### Third-Party Sharing

We do not share any data with third parties. All data stays on your local device.

### Contact

For privacy concerns, please contact via GitHub Issues.

## 한국어 (Korean)

### 데이터 수집 및 사용

WikiDocs Exporter는 사용자가 위키독스([https://wikidocs.net](https://wikidocs.net))의 컨텐츠를 로컬 노트 앱(Obsidian, Joplin)으로 내보내도록 도와주는 Chrome 확장 프로그램입니다.

### 수집하는 데이터

이 확장 프로그램은 다음 데이터를 사용자의 기기에서 로컬로 처리합니다:

- **위키독스 컨텐츠**: 위키독스 페이지의 챕터 제목, 본문, 이미지
- **사용자 설정**: 내보내기 옵션(대상 앱, 이미지 포함, 프론트매터 등)을 Chrome 로컬 스토리지에 저장
- **Joplin API 토큰**: Joplin 연동 시 사용자가 명시적으로 저장한 경우에만 로컬에 저장

### 데이터 사용 방법

- **로컬 처리만**: 모든 데이터 처리는 사용자의 기기에서 로컬로 수행됩니다. 외부 서버로 데이터가 전송되지 않습니다.
- **제3자 서비스 미사용**: 분석, 추적 또는 제3자 클라우드 서비스를 사용하지 않습니다.
- **내보내기 전용**: 데이터는 로컬 노트 앱으로 컨텐츠를 내보내는 용도로만 사용됩니다.
- **수집 않음**: 외부로 전송하거나 저장하는 개인 정보를 수집하지 않습니다.

### 데이터 저장

- 사용자 설정은 Chrome의 로컬 스토리지(`chrome.storage.local`)에 저장됩니다.
- Joplin API 토큰은 사용자의 명시적인 동의가 있는 경우에만 로컬에 저장됩니다.
- 내보낸 모든 데이터는 사용자의 로컬 기기에 유지됩니다.

### 제3자 공유

어떤 데이터도 제3자와 공유하지 않습니다. 모든 데이터는 사용자의 로컬 기기에 유지됩니다.

### 문의

개인정보 관련 우려 사항이 있으시면 GitHub Issues를 통해 문의해 주세요.
