📋 스크린샷 캡처 방법

  1. Chrome에서 WikiDocs Exporter 확장 프로그램 로드
  2. 위키독스 책 페이지 열기
  3. 확장 프로그램 아이콘 클릭 → 사이드패널 열기
  4. 각 화면에서 Cmd+Shift+S (Mac) 또는 Ctrl+Shift+S (Windows)로 스크린샷 캡처
  5. images/capture1.png ~ capture4.png로 저장
